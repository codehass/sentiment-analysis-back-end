# sentiment-analysis-back-end

This repository contains the back-end code for a sentiment analysis application. The back-end is built using fastapi , and it provides an API for analyzing the sentiment of text data.

## Features

- User-friendly interface for text input
- Real-time sentiment analysis
- Displays sentiment results with confidence scores

## Installation

Follow steps in the front-end repository to set up the application.

- Link to front-end repository: [sentiment-analysis-front-end](https://github.com/codehass/sentiment-analysis-front-end)
